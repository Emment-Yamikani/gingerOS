# gingerOs
Minimal Unix-like hobby OS
#

https://img.shields.io/tokei/lines/github/emment-yamikani/gingerOs?style=plastic

## Architecture
x86-32bit
#

## Features
- Multitasking
    - kernel threads.
    - 1:1 User threads.
- Multiprocessing
    - supports up to 8 processor cores
- Virtual Filesystem
    - ramfs called gingerfs, a minimal filesystem.
    - devfs.
    - pipefs.
#

## Build
> make
#

## How to run
> make run
#
